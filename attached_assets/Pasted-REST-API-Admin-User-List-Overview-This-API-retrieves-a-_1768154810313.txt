REST API - Admin User List
Overview
This API retrieves a list of all admin users from the system.
Authentication via Bearer Token is required.
The response returns essential profile details such as name, contact, city, region, and activation status.

Endpoint
URL
{{baseurl}}/api/rest/adminuser
HTTP Method
GET
Authentication
This API requires a valid Bearer token.

Header
Authorization: Bearer <token>
Request Parameters
This endpoint does not require any request parameters.

Success Response
Status Code: 200 OK
Response Format
An array of admin user objects.

REST API - Create Admin User
Overview
This API allows you to create a new admin user in the system.
All request parameters are mandatory.
A valid Bearer token is required for authorization.

Endpoint
URL
{{baseurl}}/api/rest/adminuser
HTTP Method
POST
Authentication
This endpoint requires a valid Bearer Token.

Headers
Authorization: Bearer <token>
Content-Type: application/json
Request Body
