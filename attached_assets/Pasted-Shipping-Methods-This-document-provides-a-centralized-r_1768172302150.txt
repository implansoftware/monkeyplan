Shipping Methods
This document provides a centralized reference for shipping methods across multiple domains.
It clearly defines which delivery options are enabled for each website.

By consolidating this data, teams can quickly validate configurations and prevent domain-level mismatches.
It also simplifies maintenance and ensures consistent order processing across all storefronts.
